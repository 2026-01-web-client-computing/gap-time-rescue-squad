# 공강구조대

공강구조대는 국민대학교 학생이 공강 시간에 이용할 수 있는 학식, 교내 시설, 학습 공간, 중앙동아리 정보를 탐색하고 현재 상황에 맞는 장소를 추천받을 수 있는 반응형 웹사이트입니다.

별도의 프레임워크나 서버 없이 HTML, CSS, Vanilla JavaScript로 구현했습니다.

## 주요 기능

### 홈

- 중식과 석식 메뉴 조회
- 메뉴 설명과 재료 정보 제공
- 재료를 바탕으로 돼지고기 포함 여부를 안내하는 로컬 식이 정보 분석 모달
- 카테고리별 교내 시설 조회
- 시설 위치, 운영시간, 상세 정보 확인
- 캠퍼스 지도에서 선택한 시설의 건물 위치 표시
- 분과별 70개 중앙동아리 조회 및 Instagram 링크 제공

> 식이 정보 분석은 외부 AI API를 호출하지 않는 클라이언트 측 데모 기능입니다.

### 공강 추천

다음 조건을 조합해 최대 5개의 장소를 추천합니다.

- 공강 시간: 30분 이하, 1시간 정도, 2시간 이상
- 활동: 쉬기, 공부하기, 밥 먹기, 카페 가기, 산책하기
- 현재 위치: 교내 주요 건물

30분 이하를 선택하면 같은 건물의 시설을 우선 탐색하며, 추천 결과에서 위치, 운영시간, 상세 정보와 추천 이유를 확인할 수 있습니다.

### 캠퍼스 시설 게임

- 시설 사진을 보고 정답을 고르는 4지선다 장소 맞히기
- 시설과 위치 정보가 맞는지 판단하는 OX 퀴즈
- 게임마다 중복 없이 무작위로 출제되는 5문제
- 문제당 10초 제한 시간
- 실시간 점수, 정답 피드백, 시설 설명 제공
- 게임 종료 후 최근 점수 표시

### 팀원 소개

- 팀 소개와 팀원별 담당 역할 제공
- 팀원 카드 클릭으로 공강 팁 표시
- `Escape` 키 또는 카드 바깥 클릭으로 열린 팁 닫기

### 다국어

공통 `assets/js/i18n.js`를 통해 다음 7개 언어의 UI 번역을 지원합니다.

- 한국어
- 영어
- 중국어
- 일본어
- 독일어
- 프랑스어
- 러시아어

모든 페이지는 `#lang-selector`와 `data-i18n` 속성을 공통으로 사용합니다. 선택한 언어는 `localStorage`에 저장되어 페이지를 이동하거나 새로고침해도 유지됩니다.

## 페이지 구성

| 파일 | 설명 |
| --- | --- |
| `main.html` | 학식, 교내 시설, 중앙동아리 탐색 |
| `team.html` | 프로젝트 및 팀원 소개 |
| `game.html` | 장소 맞히기와 OX 퀴즈 |
| `form.html` | 공강 시간, 활동, 위치 기반 장소 추천 |

모든 페이지는 공통 헤더, 네비게이션, 언어 선택기, 카드와 버튼 스타일을 공유하며 데스크톱과 모바일 화면에 대응합니다.

### JavaScript 구성

| 파일 | 역할 |
| --- | --- |
| `assets/js/data.js` | 학식, 49개 교내 시설, 70개 중앙동아리 데이터 |
| `assets/js/i18n.js` | 공통 번역 데이터, 언어 적용 및 선택 언어 저장 |
| `assets/js/app.js` | 홈 탭, 시설 지도, 동아리 필터, 식이 정보 모달 |
| `assets/js/form.js` | 조건별 시설 필터링 및 추천 결과 렌더링 |
| `assets/js/game.js` | 문제 생성, 타이머, 채점, 피드백과 화면 전환 |
| `assets/js/team.js` | 팀원 카드의 공강 팁 상호작용 |

## 실행 방법

빌드나 패키지 설치가 필요하지 않습니다.

1. 저장소를 내려받습니다.
2. 프로젝트 루트의 `main.html`을 브라우저에서 엽니다.

로컬 정적 서버를 사용해도 됩니다. 예를 들어 Python이 설치되어 있다면 프로젝트 루트에서 다음 명령을 실행합니다.

```bash
python -m http.server 8000
```

이후 브라우저에서 `http://localhost:8000/main.html`에 접속합니다.

## 데이터 및 동작 범위

- 시설, 학식, 동아리, 게임 문제 데이터는 프로젝트 내부의 정적 JavaScript 데이터입니다.
- 운영시간과 메뉴는 실시간으로 갱신되는 정보가 아니므로 실제 이용 전 학교 공식 안내를 확인해야 합니다.
- 사용자 계정, 서버 데이터베이스, 외부 API 연동은 포함하지 않습니다.
